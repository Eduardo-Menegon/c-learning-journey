## 💻 C Learning Journey

Este repositório mostra minha evolução na linguagem C, desde os conceitos básicos até estruturas de dados e algoritmos mais avançados. Aqui estão exercícios e implementações que fiz ao longo dos estudos, com foco em desenvolver lógica de programação e entender melhor como resolver problemas na prática.

## 📚 O que tem aqui?

Os estudos estão organizados em módulos para facilitar a navegação e a revisão dos conteúdos:

* **Lógica Básica:** Condicionais (if/else) e fundamentos da tomada de decisão
* **Repetição:** Uso de `for` e `while`, com foco em controle de fluxo
* **Dados:** Arrays, matrizes e uso de funções para organização do código
* **Ordenação:** Implementação de algoritmos como Bubble Sort e Selection Sort
* **Estruturas de Dados:**

  * Pilhas (Stack)
  * Filas (Queue)
  * Listas Encadeadas (Linked List)

## 🧠 Relato de Aprendizado

Aprender C não é linear. Tive momentos em que o código compilava, mas a lógica de índices em arrays me dava um nó na cabeça.

### O que mais tive dificuldade

* **Gerenciamento de índices:** Entender que um array vai de `0` até `n-1` parece simples, mas na prática um erro pode acabar com o código
* **Ponteiros:** Trabalhar com listas encadeadas exigiu uma mudança de mentalidade sobre como os dados são armazenados na memória
* **Debugging:** Uso frequente de `printf` e ferramentas de depuração para identificar falhas na lógica, especialmente em algoritmos de ordenação

### Como venho evoluindo

Tenho lidado com essas dificuldades através de prática constante: testar, errar e refazer. Quando um algoritmo não faz sentido, costumo desenhar seu fluxo no papel antes de voltar para o código. O foco está em entender o porquê das soluções, e não apenas reproduzir a sintaxe.
