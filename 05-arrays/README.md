Lógica:
Utilizei um array para armazenar vários valores inteiros e um loop `for`
para percorrer cada posição. Através do índice, consegui acessar os
elementos e realizar operações como exibição e soma dos valores.

Dificuldade:
No início, tive dificuldade em entender como o loop `for` funcionava,
principalmente o porquê de iniciar com `i = 0`. Também enfrentei dúvidas
ao trabalhar com matrizes, já que é necessário utilizar dois loops para
percorrer linhas e colunas, formando uma estrutura em 2D. Além disso,
cometi erros ao definir o limite do loop, acessando posições inválidas
do array.

Decisão:
Passei a iniciar sempre os loops em 0 para garantir o acesso correto
ao primeiro elemento do array. Também busquei entender melhor a lógica na
criação de matrizes 2D, utilizando dois loops para percorrer linhas e
colunas e respeitando suas particularidades.
